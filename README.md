# Fruit Flip Fiesta

A standalone fruit emoji memory match card game built with HTML5 Canvas and vanilla JavaScript.

## Play locally

Open `index.html` directly, or run a local server:

```bash
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173`.

## Controls

- Click/tap cards to flip and match pairs
- `Enter` to start/restart
- `R` to reset
- `F` to toggle fullscreen
